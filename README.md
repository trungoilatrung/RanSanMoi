# 🐍 GAME RẮN SĂN MỒI (JAVA)

## 📌 Giới thiệu

Đây là project xây dựng game **Rắn săn mồi** bằng ngôn ngữ **Java**, sử dụng thư viện **Java Swing** để tạo giao diện đồ họa.

Game mô phỏng trò chơi cổ điển, nơi người chơi điều khiển con rắn để ăn mồi và tăng điểm. Khi rắn va chạm vào tường hoặc chính nó thì trò chơi kết thúc.

---

## 🎮 Chức năng chính

* Di chuyển rắn bằng bàn phím (lên, xuống, trái, phải)
* Sinh mồi ngẫu nhiên
* Tăng độ dài rắn khi ăn mồi
* Tính điểm
* Tạm dừng / tiếp tục game
* Kết thúc game khi thua
* (Có thể có) lưu và tải game

---

## 🛠️ Công nghệ sử dụng

* Ngôn ngữ: **Java**
* Thư viện: **Swing / AWT**

## ▶️ Cách chạy chương trình

### Cách 1: Chạy bằng IDE

1. Mở project bằng IntelliJ IDEA hoặc Eclipse
2. Chạy file `Main.java`

### Cách 2: Chạy bằng command line

```bash
javac *.java
java Main
```
